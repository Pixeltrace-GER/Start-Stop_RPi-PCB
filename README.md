# Start & Stop HAT PCB

## Features Raspberry
-  I2C SSD1306 OLED
-  Cooling Fan 2510
-  Serial Communication [RPi <-> D1mini]
-  GCode Button 5x
-  GCode Button Led 5x

## Features D1mini
-  Start & Stop relay [HF46F/5-HS1]
-  Error LED [red 0805]
-  WiFi-AP Config
-  EEPROM saved Config
-  Serial Communication [RPi <-> D1mini]
